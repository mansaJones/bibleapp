# BibleGamifiedApp
<br>

## Project Overview
> This application is for the gaming concept with two different games 1. Tap the Matching Card & 2. Drag & Drop object
Game 1 - Tap the Matching Card

1. A player turns over any two cards (one at a time) and keeps them if the cards match (for instance, two kings or two matching numbers or alphabets)
2. If they successfully match a pair they get to keep the cards, and that player gets another turn & points will be increased.
3. When a player turns over two cards that do not match, those cards are turned face down again (in the same position) and there will be maxed 5 turns available to keep that game continue within the timer.
4. there will be some timer keep continue till when player get chances to play the game and at the end of timer player's forcefully stopped from the game and score will be declared. 
5. on the success of the current round - if the player finished it within the timer he will be allowed to move on next round where the same game with random places card will be placed with some faster timer option to increase the game complexity. Game will continue till the timer ends and move to the next round on success and exit on failed attempts 

Game 2 - Drag & Drop object
The drag and drop matching game for kids section shows a variety of shapes or animal characters as objects and makes the child drag and drop the geometric shapes in the correct position over the given image. activity will help kids to recognise the shapes & animal characters.

1. there will be 6 objects available on the screen on which users needs to drag and drop the object and place on the matching object position 
2. there will be some timer keep continue till when a player gets chances to play the game and at the end of timer player's forcefully stopped from the game and score will be declared. 
3. on the success of game 1 if the player finished it within the timer he will be allowed to move on next round where the same game with random place cards will be placed with some faster timer option to increase the game complexity. 
4. The game will continue till the timer ends and move to the next round on success and exit on failed attempts.
